Flat technical panel — hairline border, sharp corners, optional mono index. The studio's spec-sheet surface (no glass, no glow).

```jsx
<Panel number="01">
  <Eyebrow dot>Flagship</Eyebrow>
  <h3 className="cp-display" style={{fontSize:30}}>Website Design + Build</h3>
  <p>Custom builds wired for AI citation on day one.</p>
</Panel>
```

Pass `number` for the corner index, `hoverable={false}` to freeze it, `padding` to tune.
