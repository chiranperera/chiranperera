Sharp monochrome buttons, mono chips, eyebrows and status badges — the studio's technical detailing.

```jsx
<Button variant="solid" arrow>View project</Button>
<Button variant="outline">All work</Button>
<Button variant="accent" arrow>Get a free AI audit</Button>
<Eyebrow dot>Branding · Web · AI — 2026</Eyebrow>
<Tag>Next.js</Tag>
<Tag variant="accent">WebGL</Tag>
<Badge status="open">Available Q3 2026</Badge>
```

All labels are uppercase Space Mono. `Button` variants: solid (white, inverts on hover) / outline / ghost / accent (per-project colour). `Eyebrow` takes `dot` (accent) and `vertical` (side rail). `Tag` variants: default / solid / accent.
