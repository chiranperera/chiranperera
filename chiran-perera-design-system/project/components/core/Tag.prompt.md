Technical chips, eyebrows and status badges — uppercase Space Mono detailing.

```jsx
<Eyebrow dot>The studio · what I do</Eyebrow>
<Eyebrow vertical>Scroll</Eyebrow>
<Tag>Next.js</Tag>
<Tag variant="accent">AI</Tag>
<Badge status="open">Available Q3 2026</Badge>
```

`Eyebrow` — mono kicker; `dot` adds the accent dot, `vertical` rotates it for a side rail. `Tag` — chip (default hairline / solid / accent). `Badge` — status capsule with a dot.
